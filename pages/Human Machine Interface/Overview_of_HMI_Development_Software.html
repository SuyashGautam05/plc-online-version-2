Overview of HMI Development Software

  1. Siemens WinCC
    • Siemens WinCC: Siemens WinCC is a SCADA/HMI software platform from Siemens, used to monitor, control, and visualize industrial processes—especially within the Siemens automation ecosystem.
    • HMI & SCADA: Operator screens, alarms, trends, reports
    • PLC integration: Native communication with Siemens S7 PLCs (S7-300/400/1200/1500)
    • Two main forms:
      • WinCC (TIA Portal): Integrated with TIA Portal for PLC + HMI engineering
      • WinCC SCADA (Classic): Large, distributed SCADA systems
    • Security & user management: Roles, access levels, audit trails
    • Connectivity: OPC UA, industrial Ethernet, databases
    • Industries: Manufacturing, process industries, utilities, infrastructure
  2. Rockwell FactoryTalk View
    • Rockwell FactoryTalk View: Rockwell FactoryTalk View is an HMI/SCADA software suite used with Allen-Bradley PLCs to monitor, control, and visualize industrial processes.

    Key Highlights:

    • HMI & SCADA: Builds operator interfaces for machines and plants
    • PLC integration: Native communication with Allen-Bradley ControlLogix, CompactLogix, MicroLogix
    • Alarms & trends: Real-time alarms, historical data, diagnostics
    • Two main versions:
      • FactoryTalk View ME (Machine Edition): standalone machines, PanelView HMIs
      • FactoryTalk View SE (Site Edition): distributed SCADA systems
    • FactoryTalk Services: Security, user management, data access
    • Industries: Automotive, manufacturing, food & beverage, pharma
  3. Schneider EcoStruxure
    • Schneider EcoStruxure: Schneider EcoStruxure is an IIoT-enabled automation and energy management platform from Schneider Electric.
    • Three-layer architecture:
      • Connected Products: sensors, drives, breakers, PLCs
      • Edge Control: PLCs, SCADA, DCS
      • Apps, Analytics & Services: monitoring, analytics, asset management
    • SCADA & HMI: EcoStruxure Plant SCADA, Operator Terminal Expert
    • Energy management: Power monitoring, efficiency, sustainability
    • IIoT & cloud-ready: Secure connectivity, remote monitoring, analytics
    • Industries: Manufacturing, utilities, buildings, data centers, oil & gas

    It basically ties automation + power + digital intelligence into one platform.

  4. Mitsubishi GT Designer
    • Mitsubishi GT Designer: Mitsubishi GT Designer is HMI configuration software used to design and program Mitsubishi GOT touch panels.

    Here’s the essentials:

    • HMI design tool: Create operator screens
    • PLC integration: Works with Mitsubishi PLCs (FX, Q, iQ-R)
    • Screen development: Buttons, alarms, trends, recipes
    • Device monitoring: Read/write PLC data
    • Connectivity: Ethernet, serial, industrial protocols

    It’s part of GT Works3, which includes:

    • GT Designer3: HMI design
    • GT Simulator: Testing
    • GT SoftGOT: PC-based HMI
  5. Wonderware InTouch
    • Wonderware InTouch: Wonderware InTouch is an HMI software used to monitor, visualize, and control industrial processes in real time.
    • Operator interface: Graphical dashboards
    • Real-time data: Connects to PLCs, DCS
    • Alarms & trends: Displays system status
    • Industrial integration: OPC, SCADA, networks
    • Industries: Manufacturing, oil & gas, water, power, pharma
