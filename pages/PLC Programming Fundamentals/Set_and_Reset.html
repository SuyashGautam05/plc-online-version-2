Set and Reset

Set and Reset instructions are used in PLC programming to latch outputs or internal memory bits. Latching means that once an output is turned ON, it remains ON even after the activating condition is removed, until a separate reset condition is applied.

The Set (SET) instruction is used to turn ON a specific output or memory bit and keep it ON. Once the SET instruction is executed, the output remains energized regardless of changes in the input condition that activated it.

The Reset (RESET) instruction is used to turn OFF a previously set output or memory bit. The output remains OFF until it is set again by a SET instruction.

How do Set and Reset work?

Instruction Functionality Behavior
SET Turns ON a bit (sets it to 1/TRUE) Once set, the bit stays ON until reset
RESET Turns OFF a bit (resets it to 0/FALSE) Once reset, the bit stays OFF until set

Unlike normal outputs controlled directly by logic, SET/RESET latch or unlatch the output, ignoring input changes until the opposite instruction occurs.

Set and Reset instructions are commonly used in applications where memory of an event is required, such as start-stop circuits, alarm systems, fault latching, and mode selection. These instructions reduce the need for complex interlocking logic and simplify program design.

Care must be taken while using Set and Reset instructions to avoid unwanted latching conditions. Proper program structure ensures that the reset condition is always reachable and clearly defined.
