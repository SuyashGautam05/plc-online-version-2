CTD

A CTD (Counter Down) is a type of counter in PLC programming that:

  • Starts counting from a preset number (PV).
  • Counts downwards by 1 each time it receives a trigger pulse (usually a rising edge signal).
  • When the count value (CV) reaches zero, the output Q activates (turns ON).

It’s often used to track how many events or items are left until a target is reached.

Structure:

Component Meaning Role
PV (Preset Value) Starting count The initial count value loaded into the counter
CV (Current Value) Current count Decreases by 1 on each trigger
CD (Count Down Input) Input pulse The signal that triggers decrement
LD (Load Input) Load preset Loads PV into CV (resets/restarts the counter)
Q (Output) Done flag Turns ON when CV ≤ 0

Working Principle

  1. When Load (LD) is activated:
    • Preset value (PV) is loaded into CV
  2. Each rising edge at CD:
    • CV decreases by 1
  3. When CV reaches 0:
    • Output Q turns ON
  4. Counter remains at zero until reloaded
