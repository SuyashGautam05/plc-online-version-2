Hydraulic Actuator Fundamentals

What is a Hydraulic Actuator?

A hydraulic actuator converts hydraulic energy (pressurized fluid) into mechanical motion. Controlled by a PLC, it enables precise linear or rotary movement in industrial machinery.

Key principle: Pascal's Law — pressure applied at any point in a confined fluid transmits equally in all directions.

Linear Motion High Force Precise Control High Pressure

Types of Hydraulic Actuators

  • Single-Acting Cylinder — Pressure on one side only; spring or gravity returns it
  • Double-Acting Cylinder — Pressure on both sides; full bidirectional control
  • Telescopic Cylinder — Multi-stage for long stroke in short body
  • Rotary Actuator — Converts fluid pressure to rotary motion
  • Hydraulic Motor — Continuous rotation, high torque

PLC Control Principles

The PLC reads inputs (sensors, switches, HMI commands) and controls outputs (solenoid valves, pump motors) using programmed logic.

Scan Cycle: Input Read → Logic Execute → Output Write → Repeat (~10ms cycle)

EXTEND SEQUENCE
PLC energizes SOL-A → DCV shifts → Pump flow to cap end → Piston extends → PS confirms position → PLC de-energizes
RETRACT SEQUENCE
PLC energizes SOL-B → DCV shifts → Flow to rod end → Piston retracts → PS confirms → Solenoid off

Key Formulas

Force (F) = Pressure (P) × Area (A)
Flow (Q) = Area (A) × Velocity (v)
Pressure (P) = Force (F) / Area (A)
Power (W) = Pressure × Flow Rate
Bore Area (A) = π × (d/2)²

Units: P in bar, A in cm², F in kN, Q in L/min, v in m/s

Safety & Standards

  • Over-pressure protection — Relief valves prevent system damage
  • E-Stop integration — Dumps pressure on emergency stop
  • Position limits — Hard stops + limit switch redundancy
  • Lockout/Tagout — Mandatory before maintenance
  • Leak detection — Pressure drop monitoring
⚠️ Hydraulic fluid injection injuries are life-threatening. Never break connections under pressure.

PLC Ladder Logic Diagram
Ladder logic represents PLC programs visually. Each rung is a logic condition controlling an output. Click a rung to highlight it.

Ladder Logic Elements

SymbolNameDescription
—| |—NO ContactNormally Open — passes when ON
—|/|—NC ContactNormally Closed — passes when OFF
—( )—Output CoilEnergizes when rung is true
—(S)—Set CoilLatches ON until reset
—(R)—Reset CoilUnlatches a set coil
TONTimer On-DelayDelays activation

Rung Descriptions

  • Rung 1: If E-STOP not pressed AND pump start pressed → energize pump motor output
  • Rung 2: If pump running AND extend button pressed AND not at max limit → energize SOL-A
  • Rung 3: If pump running AND retract button pressed AND not at min limit → energize SOL-B
  • Rung 4: If pressure > 220 bar → set over-pressure alarm AND de-energize pump
  • Rung 5: If SOL-A AND SOL-B both ON simultaneously → fault → shut down system
Structured Text (IEC 61131-3)

Real PLC code written in Structured Text — the most versatile IEC 61131-3 language, similar to Pascal/C.

// ============================================ // Hydraulic Actuator PLC Control Program // IEC 61131-3 Structured Text // ============================================ PROGRAM HydraulicActuatorControl // Variable Declarations VAR btnExtend : BOOL; // HMI extend button input btnRetract : BOOL; // HMI retract button input btnEStop : BOOL; // Emergency stop (NC contact) lsMaxExtend : BOOL; // Limit switch: fully extended lsMaxRetract : BOOL; // Limit switch: fully retracted ptPressure : REAL; // Pressure transducer [bar] posEncoder : REAL; // Position encoder [mm] tempSensor : REAL; // Oil temperature [°C] solA : BOOL; // Output: Solenoid A (extend) solB : BOOL; // Output: Solenoid B (retract) pumpMotor : BOOL; // Output: Pump motor contactor alarmOverPressure : BOOL; alarmOverTemp : BOOL; alarmValveFault : BOOL; timerDelay : TON; // On-delay timer for pump start systemReady : BOOL := FALSE; END_VAR // ============================================ // RUNG 1: Pump Motor Control // ============================================ IF (NOT btnEStop) AND (NOT alarmOverPressure) AND (NOT alarmOverTemp) AND (btnExtend OR btnRetract) THEN pumpMotor := TRUE; timerDelay(IN := TRUE, PT := T#500ms); systemReady := timerDelay.Q; ELSE pumpMotor := FALSE; systemReady := FALSE; timerDelay(IN := FALSE); END_IF // ============================================ // RUNG 2: Extend Cylinder // ============================================ IF systemReady AND btnExtend AND NOT lsMaxExtend AND NOT solB // Interlock: prevent both on THEN solA := TRUE; ELSE solA := FALSE; END_IF // ============================================ // RUNG 3: Retract Cylinder // ============================================ IF systemReady AND btnRetract AND NOT lsMaxRetract AND NOT solA THEN solB := TRUE; ELSE solB := FALSE; END_IF // ============================================ // RUNG 4: Alarm Processing // ============================================ IF ptPressure > 220.0 THEN alarmOverPressure := TRUE; solA := FALSE; solB := FALSE; pumpMotor := FALSE; END_IF IF tempSensor > 80.0 THEN alarmOverTemp := TRUE; END_IF IF solA AND solB THEN alarmValveFault := TRUE; solA := FALSE; solB := FALSE; END_IF END_PROGRAM

IEC 61131-3 Languages

  • Ladder Diagram (LD) — Relay logic, most common
  • Structured Text (ST) — High-level, algebraic
  • Function Block Diagram (FBD) — Graphical dataflow
  • Instruction List (IL) — Assembly-like, legacy
  • Sequential Function Chart (SFC) — State machine

Key Safety Interlocks in Code

  • NOT solB in extend rung — prevents both solenoids activating simultaneously (valve conflict)
  • Pressure > 220 bar — automatic pump shutdown, prevents component damage
  • lsMaxExtend/Retract — hard limit enforcement in software before hardware limit switch
  • timerDelay — allows pump pressure to build before valve opens
Fault Diagnosis & Alarms

Click any fault card to simulate it in the system. Learn to identify symptoms and apply corrective actions.

🔴 Over-Pressure Fault
System pressure exceeds relief valve setpoint. Common cause: blocked return line, valve stuck closed, or incorrect setpoint.

Symptoms

  • Pressure gauge reads above 220 bar
  • Relief valve opens — audible hissing
  • Cylinder movement stops
  • Pump motor current spike
✅ Fix: Check return line, inspect DCV, verify relief valve setpoint
🟡 Slow Cylinder Movement
Cylinder extends/retracts slower than expected. Usually flow-related: pump wear, blocked filter, or FCV restriction.

Symptoms

  • Cycle time longer than setpoint
  • Flow meter reads below setpoint
  • Pressure normal or slightly low
  • No position errors initially
✅ Fix: Check/replace filter, inspect pump output, check FCV setting
🔵 Position Sensor Fault
PLC receives incorrect or no signal from linear position encoder. Causes erratic or no movement response.

Symptoms

  • Position reads 0% or stuck value
  • PLC fault code: I/O error
  • Cylinder may drift past limits
  • Control system goes to safe state
✅ Fix: Check wiring, sensor power supply, verify encoder calibration
🟠 Valve Solenoid Fault
Solenoid coil failure or DCV spool stuck. PLC output active but valve doesn't respond mechanically.

Symptoms

  • Solenoid LED on but no movement
  • No pressure change on actuation
  • Coil resistance out of range
  • Valve may stay in last position
✅ Fix: Measure coil resistance (typically 10–50Ω), check DCV manually
🌡️ High Oil Temperature
Oil temp exceeds 80°C. Causes fluid viscosity drop, seal degradation, and reduced lubrication effectiveness.

Symptoms

  • Temp sensor alarm active
  • System may slow or stall
  • Oil smell or discoloration
  • Seals begin to leak
✅ Fix: Check cooler/heat exchanger, reduce cycle rate, check oil level
💧 Internal Cylinder Leak
Piston seal bypassing fluid internally. Cylinder cannot hold position under load and drifts.

Symptoms

  • Cylinder drifts under load
  • Position error grows over time
  • Pressure equalization across piston
  • Lower force output than expected
✅ Fix: Replace piston seals, check for cylinder bore scoring
