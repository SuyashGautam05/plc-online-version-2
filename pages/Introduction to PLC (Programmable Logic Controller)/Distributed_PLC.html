Distributed PLC

In today’s world of advanced PLCs most modular PLCs have the features of a distributed PLC. Regardless of the terminology used a distributed PLC is universally characterized by:

  1. PLC naming convention – Distributed or Modular with Remote I/O.
  2. CPU processor – High powered.
  3. Program and data memory size – Large.
  4. Power supply – Module.
  5. Input Interface – Input Modules.
  6. Output Interface – Output Modules.
  7. Communication interfaces – Communication Modules.
  8. Mounting system – Rack, backplane, rail or chassis.
  9. Physical size – Medium to Large.
  10. Flexibility – High
  11. Expandable – Yes
  12. Customizable – Yes
  13. Applications –High end applications with large number of inputs and outputs.
