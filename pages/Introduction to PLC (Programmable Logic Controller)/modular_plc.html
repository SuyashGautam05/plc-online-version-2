Theory

A modular plc is characterized by:

  1. PLC naming convention – Modular.
  2. CPU processor – Medium to High powered.
  3. Program and data memory size – Medium to Large.
  4. Power supply – Module.
  5. Input Interface – Input Module.
  6. Output Interface – Output Modules.
  7. Communication interfaces – Communication Modules.
  8. Mounting system – Rack, backplane, rail or chassis.
  9. Physical size – Medium to Large.
  10. Flexibility – High
  11. Expandable – Yes
  12. Customizable – Yes
  13. Applications – Medium to High end applications with large number of inputs and outputs.
